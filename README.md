# SQL Data Job Analysis Project

---

## Overview

This project analyzes the **Data Jobs Market** using SQL to uncover trends in job demand, salary patterns, and skill requirements across different data-related roles such as:

- Data Analyst  
- Data Scientist  
- Data Engineer  
- Business Analyst  
- Machine Learning Engineer  

The goal of this project is to transform raw job posting data into actionable insights for both **job seekers** and **business decision-makers**.

---

## Project Goals

- Analyze the most in-demand skills in the data industry
- Compare salaries across skills and job roles
- Identify high-paying niche technologies
- Understand hiring trends in analytics and engineering
- Generate business recommendations from job market data

## Data Sources
- Job posting databases
- Salary information
- Skills requirements

##  Tools & Technologies Used

- PostgreSQL / MySQL (depending on setup)
- Joins
- Aggregate Functions
- Group By
- Subqueries
- Window Functions
- CTEs
- Data Cleaning Queries
- Data Visualization (optional dashboards)
---

## Dataset Used

The dataset contains information such as:

- Job Titles
- Skills Required
- Salary Information
- Skill Frequency
- Average Salary by Skill
- Demand Count

## Getting Started
1. Clone the repository
2. Load the datasets
3. Run SQL queries in the `scripts/` directory
4. Review findings in `analysis/`

# Key Analysis Performed
---
## 1️⃣ Most In-Demand Skills

Skills like:
- SQL  
- Python  
- R  
- Tableau  
- Power BI  

appear most frequently in job postings.

### Insight:
Core analytics skills remain highly valuable in the market.
---

## 2️⃣ Highest Paying Skills

Some niche and advanced skills offer premium salaries:
- SVN  
- Solidity  
- Couchbase  
- Golang  
- DataRobot  
- Snowflake  
- Spark  
### Insight:
Specialized technical skills often pay more than general-purpose tools.

---

## 3️⃣ Demand vs Salary Analysis

Some skills have:

- High demand + High salary → Best ROI
- High demand + Moderate salary → Safe career path
- Low demand + High salary → Niche opportunities

---

## 4️⃣ Cloud & Big Data Growth

Strong demand observed for:
- AWS  
- Snowflake  
- Redshift  
- Hadoop  
- Spark  

### Insight:
Modern businesses are investing heavily in scalable cloud data systems.
